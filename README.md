# Eastside Price Scraper

Automated price scraper for Eastern Suburbs Melbourne using GitHub Actions.

## How It Works

- **Runs every Sunday at 6 AM** (UTC)
- **Scrapes Google Maps and business websites** for current prices
- **Updates Supabase database** automatically
- **Your Loveable app** pulls live prices
